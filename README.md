# SocketChatRoom
在线多人聊天室
基于nodejs的socket.io开发

## 1. Install

```
npm install
```

## 2. Use
```
node server.js
```

## 已实现功能
* 群聊
* 选择登录头像和自定义用户名
* 短期历史记录读取（最近50条）
